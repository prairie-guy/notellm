Key Change in Altair 6.0

In Altair 6.0, the recommended way to access datasets changed:

Old (pre-6.0):

from vega_datasets import data
penguins = data.penguins()

New (6.0+):

from altair.datasets import data
penguins = data.penguins()

Important: The column names also changed from underscores to spaces:

  • Flipper_Length_(mm) Flipper Length (mm)
  • Body_Mass_(g) Body Mass (g)
  • Beak_Length_(mm) Beak Length (mm)
  • Beak_Depth_(mm) Beak Depth (mm)

Execute the cell above to load the penguin dataset and see the first few rows!

# Create violin plots for penguin measurements by species
from IPython.utils.capture import capture_output

with capture_output() as _claude_captured_output:
    # Body Mass violin plot
    chart1 = alt.Chart(penguins_pl).transform_density(
        'Body Mass (g)',
        as_=['Body Mass (g)', 'density'],
        groupby=['Species']
    ).mark_area(orient='horizontal', opacity=0.5).encode(
        y='Body Mass (g):Q',
        color='Species:N',
        x=alt.X('density:Q', stack='center', impute=None, title=None, axis=alt.Axis(labels=False, values=[0], grid=False, ticks=True)),
        column=alt.Column('Species:N', header=alt.Header(titleOrient='bottom', labelOrient='bottom'))
    ).properties(
        width=100,
        title='Body Mass by Species'
    )
    
    # Flipper Length violin plot
    chart2 = alt.Chart(penguins_pl).transform_density(
        'Flipper Length (mm)',
        as_=['Flipper Length (mm)', 'density'],
        groupby=['Species']
    ).mark_area(orient='horizontal', opacity=0.5).encode(
        y='Flipper Length (mm):Q',
        color='Species:N',
        x=alt.X('density:Q', stack='center', impute=None, title=None, axis=alt.Axis(labels=False, values=[0], grid=False, ticks=True)),
        column=alt.Column('Species:N', header=alt.Header(titleOrient='bottom', labelOrient='bottom'))
    ).properties(
        width=100,
        title='Flipper Length by Species'
    )
    
    # Combine charts vertically
    combined = alt.vconcat(chart1, chart2)
    display(combined)

for output in _claude_captured_output.outputs:
    display(output)

Claude: The idiomatic way to create violin plots with seaborn is much simpler! Seaborn has a dedicated violinplot() function:

 CreateNotebookCell

Claude: I've created a seaborn version that's much more concise! The key differences:

Seaborn's idiomatic approach:

  • Direct sns.violinplot() function - no density transforms needed
  • Simple syntax: just specify data, x, y, and hue parameters
  • Works seamlessly with Polars DataFrames (seaborn automatically handles them)
  • Built-in styling with sns.set_theme()

This is why seaborn is popular for statistical visualizations - what takes many lines in Altair is just one line with seaborn! Execute the cell to see the violin plots.

In[10]:
import seaborn as sns
import matplotlib.pyplot as plt
from IPython.utils.capture import capture_output

with capture_output() as _claude_captured_output:
    # Set the style
    sns.set_theme(style="whitegrid")
    
    # Create a figure with two subplots
    fig, axes = plt.subplots(1, 2, figsize=(14, 6))
    
    # Body Mass violin plot
    sns.violinplot(data=penguins_pl, x='Species', y='Body Mass (g)', 
                   hue='Species', ax=axes[0], legend=False)
    axes[0].set_title('Body Mass by Species')
    
    # Flipper Length violin plot
    sns.violinplot(data=penguins_pl, x='Species', y='Flipper Length (mm)', 
                   hue='Species', ax=axes[1], legend=False)
    axes[1].set_title('Flipper Length by Species')
    
    plt.tight_layout()
    plt.show()

for output in _claude_captured_output.outputs:
